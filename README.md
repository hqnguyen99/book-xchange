# Book-Xchange

Book-Xchange is an open-source platform that allows buy or sell second-hand books

## Requirements
Docker   
yarn

## Installation with Docker

```bash
docker-compose up --build
```
Website will be available on localhost:3000

## Installation with Docker
```
yarn install
yarn start
```
Website will be available on localhost:3000
